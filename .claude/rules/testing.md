# rules/testing.md

## 1. 手動テスト（毎回やる最低ライン）

- [ ] 新規 Blender 起動 → PME2 有効化でエラーが出ない
- [ ] 既存の Pie Menu を 1 つ呼び出して動くこと
- [ ] 新規 Pie Menu を 1 つ作成
      - 項目を 1 つ追加
      - Blender を再起動しても残っていること

## 2. 追加でやるテスト

- 設定フォーマットや JSON を触ったとき:
  - [ ] JSON エクスポート → インポートで往復できること

## 3. 自動テスト（将来）

- core 層（bpy 非依存）ができたら pytest を導入する
- それまでは「自動テストなし」でよい
